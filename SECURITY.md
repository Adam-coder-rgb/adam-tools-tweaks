# Security Policy

## Reporting a Vulnerability

We take the security of our tools seriously. If you have discovered a potential security vulnerability in any of our scripts, please report it to us privately so we can investigate and fix it without exposing the community to risk.

### How to report:
Please send an email to: **adamisniceandmean@gmail.com**

Include the following information in your report:
* **The script name or tool affected.**
* **A clear description of the vulnerability.**
* **Steps to reproduce the issue.**

### What to expect:
* **Acknowledgment:** You will receive a response within 48 hours.
* **Investigation:** We will assess the issue and work on a fix.
* **Disclosure:** Once a fix is verified, we will update the repository and credit you for the discovery (if desired).

**Please do not open a public issue on GitHub for security vulnerabilities.** Publicly disclosing vulnerabilities can put our users at risk.
