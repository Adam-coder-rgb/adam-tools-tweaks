# Contributing to Adam's Tools and Tweaks

Thank you for your interest in improving this project! Whether you're reporting a bug, suggesting a feature, or submitting a pull request, your help is appreciated.

## How to Help

### Reporting Bugs
If you find a bug, please open an [Issue](https://github.com/7advamz/adams-tools-tweaks/issues) and include:
* A clear description of the issue.
* The version of PowerShell you are running.
* Steps to reproduce the behavior.

### Suggesting Features
If you have an idea for a new tool or an improvement, open an Issue labeled "Feature Request" and describe the use case.

### Pull Requests
1. Fork the repository.
2. Create your feature branch (`git checkout -b feature/AmazingTool`).
3. Commit your changes.
4. Push to the branch (`git push origin feature/AmazingTool`).
5. Open a Pull Request.

## Code Style
Please keep your PowerShell code consistent:
* Use standard naming conventions.
* Include comments for complex logic.
* Ensure your scripts are modular.
