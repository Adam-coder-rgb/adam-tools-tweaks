# Adam's Tweaks & Tools

A professional system optimization suite for Windows 10/11, featuring a high-fidelity Command Line Interface (CLI) and modular utility management.

## Overview
Adam's Tweaks & Tools is designed to streamline system maintenance and optimization tasks through a unified, professional-grade interface. By leveraging a modular architecture, it provides a stable environment to execute power-user scripts and system tools with ease.

## Requirements
* **OS:** Windows 10 / 11
* **Permissions:** Administrator privileges are required to modify system settings and registry keys.

## Quick Start
1. Extract the project folder to your desired location.
2. Press `Win + R`, type `cmd.exe`, and press `Ctrl + Shift + Enter` to launch an elevated Command Prompt.
3. Navigate to the directory: `cd C:\Users\YOUR_USERNAME\Downloads\Adam.s.Tweaks.Tools.v1.1.0\`
4. Run `.\launcher.cmd`.

## Troubleshooting
If the window fails to launch, ensure you are running the command prompt with elevated (Administrator) privileges. For persistent issues, verify the script path and your current directory permissions.
