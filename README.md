# Adam's Tool Launcher

Adam's Tool Launcher is a robust, modular, and portable Windows management suite designed for power users. It simplifies system maintenance, performance optimization, and diagnostics through an easy-to-use Command Line Interface.

## Features

### System Tweaks (1-9)
Automated registry and system configuration changes to improve performance and usability, including:
- Cleanup of temporary system files.
- Customization of the Windows 11 UI (Context Menu).
- Privacy optimizations (Search/Tips/Game Bar).
- Performance tuning (Shutdown/Fast Startup).

### Utility Menu (T)
A comprehensive diagnostic and utility hub:
- **System Info**: Hardware and OS summary.
- **Network**: DNS Cache management and active connection monitoring.
- **Performance**: Process management and System Uptime.
- **Storage**: Real-time disk space reporting.
- **Admin**: Quick access to Task Manager, Registry Editor, and Windows Update.

## How to Use
1. Download the latest release from the repository.
2. Extract the files to your preferred location.
3. Run `launcher.cmd` as Administrator (The script will automatically request elevation).
4. Follow the on-screen menu to select your desired tweak or utility.
